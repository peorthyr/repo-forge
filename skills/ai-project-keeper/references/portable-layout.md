# Portable Layout for ai-project-keeper

Recommended structure:

- SKILL.md
- references/
- assets/templates/
- evals/
- scripts/

## References
Keep procedural guidance modular and on-demand.

## Templates
Keep repeatable report and decision structures in assets/templates.

## Evals
Store prompt suites for review/update/cleanup scenarios.

## Scripts
Automate repetitive eval scaffolding.

## Portability rules
- Use relative links from SKILL.md.
- Avoid absolute path assumptions.
- Keep cleanup policies explicit and reversible.
- Track deferred work in backlog templates.
