# Portable Layout for ai-project-seed

Recommended structure:

- SKILL.md
- references/
- assets/templates/
- evals/
- scripts/

## References
Documentation loaded on demand.

## Assets/Templates
Reusable text structures for repeatable outputs.

## Evals
Prompt set and optional objective checks.

## Scripts
Helper automation for repetitive setup tasks.

## Portability rules
- Use relative links from SKILL.md.
- Keep files tool-agnostic and reusable.
- Avoid workspace-specific absolute paths.
- Keep defaults minimal and easy to override.
